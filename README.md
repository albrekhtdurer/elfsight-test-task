# Elfsight Test Task (Rick and Morty API)

SPA-приложение тестового задания для Elfsight - галерея персонажей Rick and Morty с возможностью применения фильтров для поиска. Приложение доступно по адресу: https://albrekhtdurer.github.io/elfsight-test-task/

## Запуск проекта

* Склонировать проект на ветке `main`
* Установить зависимости `npm install`
* Запустить проект командой `npm run start`. Приложение будет доступно по адресу `localhost:3000`.

## Особенности проекта

Исправлены ошибки в стартер-ките:
* ошибки и предупреждения eslint
* При открытии попапа в нём отображается бесконечный лоадер
* Не отображается Gender иконка возле имени персонажа
* При клике на First и Last страницу в списке страниц отображалась ошибка, т.к. запрашивался не тот номер страницы

Добавлены компоненты текстового инпута и дропдауна, а также компонент формы для добавления на страницу фильтров согласно [макету](https://www.figma.com/design/Ho8OTnjk2vs8WEQhL6StI6/Filters-for-Rick-and-Morty-App). Опции для дропдаунов (Gender, Status, Species) добавлены в репозиторий как константы.
